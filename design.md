# Essensplan Design Direction

## Visual Style
- Minimal productivity app, not a lifestyle landing page.
- German labels, compact controls, high scanability, and restrained visual hierarchy.
- Palette: warm paper background, ink text, muted green, tomato accent, calm blue, and soft amber status tones.

## Layout
- App shell uses a left navigation rail on desktop and a top tab strip on small screens.
- Main views use dense but readable cards with 8px radius.
- Dashboards emphasize status, next actions, and weekly context.
- Calendar and grocery grids must be responsive without overlapping text.

## Components
- Use familiar icon buttons for navigation and actions where appropriate.
- Numeric fields use steppers or standard number inputs.
- Binary item state uses checkboxes.
- Option sets use segmented controls or select inputs.

## Accessibility
- Maintain visible focus rings and semantic headings.
- All form controls need labels.
- Tables and dense lists need clear row labels.
- Do not rely on color alone for nutrition or shopping status.
