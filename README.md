# Essensplan

Essensplan ist eine deutsche, lokale Web-App für Wochenplanung, Nährwerte, Vorrat und einkaufslistenbasierte Planung.

## Funktionen
- Wochenplan mit Frühstück, Mittag, Abendessen und Snacks
- Rezeptverwaltung mit Zutaten und Nährwerten pro Portion
- Lokale Speicherung im Browser mit IndexedDB/Dexie
- Vorratsabzug und generierte Einkaufsliste
- Preisbeobachtungen je Geschäft
- Open-Food-Facts-Adapter ohne API-Key

## Lokal starten
```bash
npm install
npm run dev
```

Danach die angezeigte lokale URL im Browser öffnen.

## Qualität
```bash
npm run test
npm run build
npm run e2e
```
